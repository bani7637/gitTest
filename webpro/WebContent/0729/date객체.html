1. Date객체 생성

-new Date() >> 현재날짜 찍힘
-new Date(98765678) >> 1970.1.1기준으로 경과한(초) 날짜 찍힘
-new Date("2010.02.23")
-new Date(2000, 02, 23, 12,30, 12)>>숫자로 직접줄때는 월에 -1해줘야함
=>0월부터시작, 숫자로Date 객체생성시 하나 작은 값으로 설정한다.

2. Date객체 생성

-now = new Date()
-now.toLocaleString() : 날짜와 시간
-now.toLocaleDateString() : 날짜
-now.toLocaleTimeString() : 시간

3. Date객체 요일구하기

-now = new Date()
-now.getDay() : 0(일요일)~6(토요일)숫자로 나옴

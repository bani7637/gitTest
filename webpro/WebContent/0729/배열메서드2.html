1. slice/splice

-slice(start, end) : start~end-1 까지의 요소를 추출
-splice(index, n, a,b,c) : index위치에서 n개를 삭제하고, a,b,c요소를 추가한다.


2. concat/join/reverse

-a.concat : 여러개의 배열요소를 한개의 배열로 합친다.
-a.join : 한개의 배열요소를 하나의 문자열로 변환한다.
=>구분자를 지정할 수 있으며 생략시 콤마로 구분한다.

-a.reverse : a배열의 순서를 뒤깁기

3. sort

arr = [234, 21, 150, 11, 55, 501]
-a.sort() : 11, 150, 21, 234, 501, 55 첫째자리로 정렬, 첫째자리가 같으면 둘째자리로 정렬

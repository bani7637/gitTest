	의사 클래스(pseudo-class, 가상클래스):  
	클래스가 정의된 것처럼 간주. 
	상태나 구조에 의해서 선택이 이루어진다 
	
	상태 {}안에 내용은 기본값
	a:link { color: blue; } => 처음상태
	a:visited { color: green; } => 한번 방문한 상태
	a:hover { color: red; } => 마우스가 올려있는 상태
	a:active {color : pink; } => 마우스가 down된 상태
